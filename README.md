# 📦 Installation

```bash
git clone https://github.com/gunduul/shelbychat.git
cd shelbychat
npm install

# Environment Variables
echo "GROQ_API_KEY=your_key_here" > .env
echo "HF_API_KEY=your_key_here" >> .env

# Run locally
vercel dev
```
